# 15yan
[十五言]轻量级客户端， 提供基本的浏览功能

主要练习[square][1]公司的一些开源组件, 大爱其风格


use android studio 1.1.0

用到的依赖：

* [ollie][2]
* [dagger][3]
* [picasso][4]
* [retrofit][5]
* [ultra-ptr][6]
* [roundedimageview][7]
* [observablescrollview][8]
* [swipeback][9]
* [material-ripple][10]
